# 🎥 AZ Video Chat App

Ye ek **Real-time Video Calling aur Chat Application** hai jo maine React.js aur Node.js use karke banayi hai.
Isme aap doston ke saath video call kar sakte hain, chat kar sakte hain, aur photos bhi share kar sakte hain!

## 🔴 Live Demo
[👉 Click Here to Use App](https://az-chat.vercel.app/)

## ✨ Features
* **📞 Video Call:** HD Quality real-time calling.
* **💬 Live Chat:** Bina page refresh kiye messages bhejein.
* **📸 File Sharing:** Chat mein photos send karein.
* **📱 Responsive:** Mobile aur Laptop dono par chalta hai.
* **🔳 Floating Chat:** Video dekhte hue chat karne ka floating window.

## 🛠️ Tech Stack
* **Frontend:** React.js, HTML5, CSS3
* **Backend:** Node.js, Express.js
* **Real-time:** Socket.io, SimplePeer (WebRTC)
