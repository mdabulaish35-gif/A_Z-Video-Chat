# 🎥 AZ Video Chat App

This is **Real-time Video Calling and Chat Application** that i made for real time chating and video calling with your friends and families, and also share photos!

## 🔴 Live Demo
[👉 Click Here to Use App](https://az-chat.vercel.app/)

## ✨ Features
* **📞 Video Call:** HD Quality real-time calling.
* **💬 Live Chat:** live chatting while video call.
* **📸 File Sharing:** send photos and files in chat.
* **📱 Responsive:** runs on phone and laptop.

## 🛠️ Tech Stack
* **Frontend:** React.js, HTML5, CSS3
* **Backend:** Node.js, Express.js
* **Real-time:** Socket.io, SimplePeer (WebRTC)
