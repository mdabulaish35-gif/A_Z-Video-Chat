## 📖 About The Project

**AZ Video Chat App** is a modern communication tool designed to connect people instantly. By leveraging WebRTC for peer-to-peer media streaming and Socket.io for real-time signaling, this application provides a smooth, low-latency environment for face-to-face conversations and live text chat.

Whether you're on a laptop or a mobile device, AZ Video Chat ensures you stay connected without compromising on performance or user experience.

## ✨ Key Features

* 📞 **High-Definition Video Calling:** Crystal-clear, peer-to-peer video and audio communication.
* 💬 **Real-Time Live Chat:** Lightning-fast, bi-directional text messaging without any page reloads.
* 📸 **Media & File Sharing:** Effortlessly send photos and files directly within the active chat window.
* 🔳 **Floating Chat Interface:** Multitask like a pro! A convenient floating chat window allows you to text seamlessly without obstructing your video feed.
* 📱 **Fully Responsive Design:** A carefully crafted UI that adapts perfectly to desktops, tablets, and smartphones.

## 🛠️ Technology Stack

This application is built utilizing modern web technologies to ensure scalability, speed, and reliability.

**Client-Side (Frontend):**
* React.js
* HTML5 & CSS3
* SimplePeer (WebRTC wrapper)

**Server-Side (Backend):**
* Node.js
* Express.js
* Socket.io (Real-time event-based communication)
